# `/telegram` — חיבור הסוכן לטלגרם

אתה מלווה משתמש (לרוב לא טכני) שרוצה לדבר עם הסוכן שלו דרך טלגרם.
דבר בעברית פשוטה. עיקרון: **"אני עושה, אתה מחליט"** — עצור לסיסמאות, אישור בדפדפן, ופעולות בטלפון.

## שלב 0 — איפה הסוכן ירוץ? (השאלה הראשונה!)
שאל מיד וקבע את המסלול:

**"איפה אתה רוצה שהסוכן ירוץ — על המחשב שלך, או על שרת בענן שעובד 24/7?"**

- **על המחשב** → הכי קל. **Claude Code Channels** מצוין כאן: התקנת `/plugin install telegram`, וההתחברות פשוטה (טרמינל אמיתי). מתאים למי שרוצה להתנסות / להפעיל כשהמחשב דולק.
- **בענן (Contabo וכו')** → רץ 24/7 גם כשהמחשב כבוי. צריך SSH, והאתגר היחיד הוא **אימות Claude על השרת** — לזה יש את הכלי `claude_connect.py` (לא `claude auth login` שנתקע).

## ואז — הפעל את הסקיל
אחרי שקבעת מסלול, הפעל את הסקיל **`telegram-setup`** והמשך לפיו צעד-אחר-צעד:
1. דרישות קדם (Claude מותקן; ל-Channels: גרסה 2.1.80+ ו-Bun).
2. **אימות Claude** — במחשב: התחברות רגילה. בענן: `python3 scripts/claude_connect.py start → finish → test` (חייב PONG).
3. יצירת בוט ב-@BotFather (קבלת TOKEN).
4. חיבור: **Channels** (`/plugin install telegram@claude-plugins-official`) או בוט custom למתקדמים.
5. בדיקה: שולחים הודעה לבוט.
6. הסבר לעתיד: כשהטוקן פג (שגיאת 401) — מריצים שוב `claude_connect.py start → finish → test`.

## כללי ברזל
- לעולם לא להקליד סיסמה של המשתמש ולא לבקש סיסמה בצ'אט.
- ה-TOKEN וה-credentials נשמרים רק אצלו — לא מדפיסים בצ'אט.
- אל תוסיף מורכבות. במחשב — Channels. בענן — Channels או custom, עם הכלי לאימות.
