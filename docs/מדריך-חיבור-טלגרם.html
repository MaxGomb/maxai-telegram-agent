שתי דרכים — בחר אחת

Channels מומלץ

תוסף מובנה של Claude Code. בלי לכתוב קוד. שולח הודעה → Claude Code עצמו עונה, עם כל הכלים והזיכרון. הכי קל.

בוט Python משלך מתקדם

ערכת 12-telegram-bot. שליטה מלאה והתאמה אישית — אבל יותר שלבים. בחר רק אם אתה רוצה בוט מותאם.

הדבר היחיד שמבלבל את כולם זה לא טלגרם — זה לאמת את Claude על השרת (שלב 3), וזה שהאימות פג כל כמה חודשים (שלב 7). המדריך הזה פותר בדיוק את שני אלה. עקוב לפי הסדר.

1דרישות קדם

הכל רץ על השרת שלך. ודא שיש:

  • Claude Code מותקן ומחובר למנוי שלך (Max/Pro). בדיקה: claude --version
  • לדרך ה-Channels: גרסה 2.1.80 ומעלה, ו-Bun מותקן.
  • חשבון טלגרם פעיל.

התקנת Bun (אם אין). בטרמינל של השרת:

curl -fsSL https://bun.sh/install | bash

סגור ופתח טרמינל חדש, ואז בדוק: bun --version

בדיקה מהירה של הכל בבת אחת — מהתיקייה הזו על השרת:

bash scripts/connect_telegram.sh

הסקריפט בודק גרסאות, Bun, והכי חשוב — אם Claude מאומת, ואומר לך מה חסר.

2יצירת בוט בטלגרם (BotFather)

  1. פתח טלגרם, חפש @BotFather ופתח אותו.
  2. שלח /newbot.
  3. תן שם לבוט (למשל "MaxAI Assistant").
  4. תן שם-משתמש שמסתיים ב-bot (למשל maxai_assistant_bot).
  5. קבל את ה-TOKEN — מחרוזת בסגנון 1234567890:ABCdef.... שמור אותו.
שמור על ה-TOKEN בסוד. מי שיש לו את ה-TOKEN יכול לשלוט בבוט. לא משתפים אותו ולא מעלים אותו לאינטרנט.

3אימות Claude על השרת — הצעד הקריטי

בשתי הדרכים, Claude חייב להיות מאומת על השרת. אל תשתמש ב-claude auth login הרגיל — מסך ההתחברות שלו דורש מסך אמיתי ונתקע על שרת מרוחק. במקום, השתמש בכלי המצורף:

3.1 · מייצרים לינק התחברות

python3 scripts/claude_connect.py start

יודפס לינק. פתח אותו בדפדפן (בכל מכשיר), התחבר עם המנוי שלך, ואשר. תקבל קוד בסגנון xxxx#yyyy — העתק אותו.

3.2 · מדביקים את הקוד ומסיימים

python3 scripts/claude_connect.py finish 'הקוד-שהעתקת'

3.3 · בדיקה חיה (חובה!)

python3 scripts/claude_connect.py test
אם חזר PONG — Claude מאומת ועובד. אפשר להמשיך. 🎉
📸 מקום לצילום מסך — פלט הטרמינל של claude_connect.py test עם ה-PONG הירוק.
(להוספה: שים את התמונה ב-images/ וקרא לה auth_test_pong.png)
למה זה כך? Claude מתחבר עם מנוי (לא API key בתשלום). הטוקן מתחדש לבד — עד שיום אחד הוא פג, ואז רואים שגיאת 401. שלב 7 מסביר איך מתקנים בדקה.

4דרך A — Channels (המומלצת)

בתוך Claude Code על השרת (הרץ claude), כתוב:

/plugin install telegram@claude-plugins-official

אם מתקבלת שגיאה "לא נמצא", הוסף קודם את המרקטפלייס:

/plugin marketplace add anthropics/claude-plugins-official

עקוב אחרי ההגדרה — בשלב מסוים תתבקש להדביק את ה-TOKEN מ-BotFather (שלב 2).

5דרך B — בוט Python משלך (למתקדמים)

אם בחרת בבוט מותאם במקום Channels — השתמש בערכת 12-telegram-bot:

  1. העתק .env.example ל-.env.
  2. הכנס את TELEGRAM_BOT_TOKEN (מ-BotFather) ואת TELEGRAM_CHAT_ID (מ-@userinfobot).
  3. הרץ את הבוט, ופרוס כשירות systemd כדי שירוץ 24/7.

הוראות מלאות בתוך הערכה. (גם דרך זו דורשת שלב 3 — אימות Claude.)

6בדיקה — שלח הודעה לבוט

פתח את הבוט שלך בטלגרם ושלח לו הודעה (למשל "היי"). הוא אמור לענות.

7כשהבוט מפסיק פתאום — תפוגת טוקן (401)

כל כמה חודשים הטוקן של Claude פג, והבוט יחזיר שגיאה (לרוב מוזכר בה 401 / authentication). זה נורמלי ולוקח דקה לתקן — בדיוק כמו שלב 3:

python3 scripts/claude_connect.py start     # פתח את הלינק, אשר, קח קוד
python3 scripts/claude_connect.py finish 'הקוד'
python3 scripts/claude_connect.py test      # ודא PONG
לבדוק את מצב הטוקן בכל רגע: python3 scripts/claude_connect.py check — מראה מתי הטוקן פג.
